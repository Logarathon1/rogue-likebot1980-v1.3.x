// How to run //
// Run the run-rpgbot.bat file to start game //
// Use numbers to execute reacts //
// Angry = 1 //
// Like = 2 //
// Haha = 3 //
// Wow = 4 //
// Love = 5 //
// Sad = 6 //

// Console Commands list //
// Run the run-rpgbot.console.bat file to start the command console //
// !player = returns the player //
// !enemy = returns the enemy //
// !dungeon = returns the layout of The Dungeon //
// !stage = returns the current stage //
// !floor = returns the current floor //
// !room = returns the current room //
// !progress = returns the player's progress //
// !deathlog = returns the deathlog //
// !deathlog!# = returns the numbered deathlog entry //
// !randplayer = generates a random explorer sprite, saved in the bin folder as randplayer.png and randplayer_aura.png //

// Run is saved in savefile.json //
// To reset a run replace savefile.json with template-savefile2.json //

// All Facebook functionality has been commented out, runs solely in console //

// Enable sprite generation by replacing var outputImages = false to var outputImages = true //
// Sprites are saved in the bin folder as output, player, and player_aura //

// Will have bugs, when inputting to the console be sure to use the correct command/number //
